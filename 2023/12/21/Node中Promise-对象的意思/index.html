NodeдёӯPromise еҜ№иұЎзҡ„ж„ҸжҖқ

жӣҙж–°: 2023-12-22 13:08:17

NodeдёӯPromise еҜ№иұЎзҡ„ж„ҸжҖқ

Promise еҜ№иұЎжҳҜ JavaScript дёӯеӨ„зҗҶејӮжӯҘж“ҚдҪңзҡ„дёҖз§Қж–№ејҸпјҢе®ғд»ЈиЎЁдәҶдёҖдёӘејӮжӯҘж“ҚдҪңзҡ„жңҖз»Ҳе®ҢжҲҗжҲ–еӨұиҙҘпјҢе№¶еҸҜд»Ҙиҝ”еӣһдёҖдёӘз»“жһңжҲ–й”ҷиҜҜгҖӮ

Promise зҡ„еҹәжң¬жҰӮеҝө

  • Promise жҳҜдёҖдёӘжһ„йҖ еҮҪж•°
    • жҲ‘д»¬еҸҜд»ҘеҲӣе»әPromiseзҡ„е®һдҫӢ const p= new Promise()
    • new еҮәжқҘзҡ„дёҖдёӘе®һдҫӢеҜ№иұЎпјҢд»ЈиЎЁдёҖдёӘејӮжӯҘж“ҚдҪң
  • Promise.prototypeдёҠеҢ…еҗ«then()
  • .thenпјҲпјүж–№жі•з”ЁжқҘйў„е…ҲжҢҮе®ҡжҲҗеҠҹе’ҢеӨұиҙҘзҡ„еӣһи°ғеҮҪж•°
    • p.then(жҲҗеҠҹеӣһи°ғеҮҪж•°пјҢеӨұиҙҘзҡ„еӣһи°ғеҮҪж•°)
    • p.then(result=>{},error=>{})

Promise еҜ№иұЎжңүдёүз§ҚзҠ¶жҖҒпјҡ

  1. PendingпјҲиҝӣиЎҢдёӯпјүпјҡеҲқе§ӢзҠ¶жҖҒпјҢиЎЁзӨәејӮжӯҘж“ҚдҪңиҝҳеңЁиҝӣиЎҢдёӯпјҢж—ўдёҚжҳҜжҲҗеҠҹд№ҹдёҚжҳҜеӨұиҙҘзҠ¶жҖҒгҖӮ
  2. FulfilledпјҲе·Іе®ҢжҲҗпјүпјҡиЎЁзӨәејӮжӯҘж“ҚдҪңжҲҗеҠҹе®ҢжҲҗпјҢе№¶иҝ”еӣһдәҶдёҖдёӘз»“жһңгҖӮ
  3. RejectedпјҲе·ІеӨұиҙҘпјүпјҡиЎЁзӨәејӮжӯҘж“ҚдҪңеӨұиҙҘпјҢе№¶иҝ”еӣһдәҶдёҖдёӘй”ҷиҜҜгҖӮ

Promise еҜ№иұЎзҡ„жһ„йҖ еҮҪж•°жҺҘеҸ—дёҖдёӘжү§иЎҢеҷЁеҮҪж•°дҪңдёәеҸӮж•°пјҢиҝҷдёӘжү§иЎҢеҷЁеҮҪж•°жңүдёӨдёӘеҸӮж•°пјҢеҲҶеҲ«жҳҜ resolve е’Ң reject еҮҪж•°гҖӮеңЁжү§иЎҢеҷЁеҮҪж•°дёӯпјҢжҲ‘д»¬еҸҜд»Ҙжү§иЎҢејӮжӯҘж“ҚдҪңпјҢе№¶еңЁйҖӮеҪ“зҡ„ж—¶еҖҷи°ғз”Ё resolve жҲ– reject еҮҪж•°жқҘж”№еҸҳ Promise еҜ№иұЎзҡ„зҠ¶жҖҒгҖӮ

Promise еҜ№иұЎе…·жңүд»ҘдёӢзү№зӮ№пјҡ

  1. Promise еҜ№иұЎжҳҜдёҚеҸҜеҸҳзҡ„пјҢдёҖж—ҰзҠ¶жҖҒж”№еҸҳе°ұж— жі•еҶҚж¬Ўж”№еҸҳгҖӮ
  2. Promise еҜ№иұЎеҸҜд»ҘйҖҡиҝҮ .then() ж–№жі•ж·»еҠ жҲҗеҠҹзҠ¶жҖҒзҡ„еӣһи°ғеҮҪж•°пјҢйҖҡиҝҮ .catch() ж–№жі•ж·»еҠ еӨұиҙҘзҠ¶жҖҒзҡ„еӣһи°ғеҮҪж•°пјҢд№ҹеҸҜд»ҘдҪҝз”Ё .finally() ж–№жі•ж·»еҠ ж— и®әжҲҗеҠҹжҲ–еӨұиҙҘйғҪдјҡжү§иЎҢзҡ„еӣһи°ғеҮҪж•°гҖӮ
  3. Promise еҜ№иұЎеҸҜд»ҘйҖҡиҝҮ Promise й“ҫе®һзҺ°еҜ№еӨҡдёӘејӮжӯҘж“ҚдҪңзҡ„дёІиЎҢжҲ–е№¶иЎҢеӨ„зҗҶгҖӮ
  4. Promise еҜ№иұЎеҸҜд»ҘйҖҡиҝҮ async/await иҜӯжі•иҝӣиЎҢжӣҙз®ҖжҙҒзҡ„ејӮжӯҘж“ҚдҪңеӨ„зҗҶгҖӮ

еҸҜд»Ҙз”Ёconsole.dir(Promise) жқҘжҹҘиҜҫPromiseеҜ№иұЎзҡ„еұһжҖ§

дҪҝз”Ё Promise еҜ№иұЎеҸҜд»ҘжӣҙеҘҪең°еӨ„зҗҶејӮжӯҘж“ҚдҪңпјҢйҒҝе…ҚдәҶеӣһи°ғең°зӢұе’ҢеұӮеұӮеөҢеҘ—зҡ„й—®йўҳпјҢдҪҝд»Јз ҒжӣҙеҠ жё…жҷ°е’ҢеҸҜз»ҙжҠӨгҖӮ
console.dir жҳҜ JavaScript дёӯзҡ„дёҖдёӘж–№жі•пјҢз”ЁдәҺе°ҶдёҖдёӘеҜ№иұЎзҡ„жүҖжңүеҸҜжһҡдёҫеұһжҖ§жү“еҚ°еҲ°жҺ§еҲ¶еҸ°дёӯпјҢд»ҘдҫҝжҹҘзңӢеҜ№иұЎзҡ„з»“жһ„е’ҢеұһжҖ§гҖӮ

иҜҘж–№жі•жҺҘеҸ—дёҖдёӘеҜ№иұЎдҪңдёәеҸӮж•°пјҢе№¶е°ҶеҜ№иұЎзҡ„еұһжҖ§д»Ҙй”®еҖјеҜ№зҡ„еҪўејҸжү“еҚ°еҲ°жҺ§еҲ¶еҸ°дёӯгҖӮдёҺ console.log() ж–№жі•дёҚеҗҢпјҢconsole.dir() ж–№жі•дјҡжҳҫзӨәеҜ№иұЎзҡ„еұһжҖ§зҡ„иҜҰз»ҶдҝЎжҒҜпјҢеҢ…жӢ¬еұһжҖ§еҗҚз§°гҖҒеұһжҖ§еҖје’ҢеұһжҖ§зҡ„ж•°жҚ®зұ»еһӢгҖӮ

console.dir() ж–№жі•еңЁи°ғиҜ•е’ҢејҖеҸ‘иҝҮзЁӢдёӯйқһеёёжңүз”ЁпјҢеҸҜд»Ҙеё®еҠ©ејҖеҸ‘дәәе‘ҳдәҶи§ЈеҜ№иұЎзҡ„з»“жһ„е’ҢеұһжҖ§пјҢд»ҘдҫҝжӣҙеҘҪең°зҗҶи§Је’Ңи°ғиҜ•д»Јз ҒгҖӮ
