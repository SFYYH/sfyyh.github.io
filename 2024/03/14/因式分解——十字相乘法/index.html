因式分解——十字相乘法

2024-03-14 考研数学 因式分解, 数学 0 评论 字数统计: 1.1k(字) 阅读时长: 4(分)

十字相乘法是一个非常有用的方法，但是它有一定的限制条件，十字相乘法必须是二次三项式才可以用。

十字相乘法

1. 根据多项式的乘法法则我们可以得到： (x+a)(x+b)=x2+(a+b)x+ab(x+a) (x+b) = x ^2 + (a+b) x+ab(x+a) (x+b) = x ^2 + (a+b) x+ab ．

利用这个规律，我们可以得到分解形如 x2+px+q x ^2 + px+q x ^2 + px+q 的二次三项式的方法：如果可以找到两个数 a、b，使得常数项为两者的积，同时一次项系数为两者的和，即 ab=q ab = q ab = q ， a+b=pa+b = pa+b = p ，则 x2+px+q=(x+a)(x+b) x ^2 + px+q = (x+a) (x+b) x ^2 + px+q = (x+a) (x+b) ．

这种分解方法可以用上面的十字表示，其中：

第一列的积 1×1=11×1 = 11×1 = 1 为二次项系数；

第二列的积 a·b=q a · b = q a · b = q 为常数项；

列间的交叉乘积和 1·b+1·a=a+b=p1 · b+1 · a = a+b = p 1 · b+1 · a = a+b = p 为一次项系数．

2. 这个方法可以推广到二次项系数不为 1 的情形：

根据多项式的乘法法则我们可以得到： (ax+b)(cx+d)=acx2+(ad+bc)x+bd(ax+b) (cx+d) = acx^2 + (ad +bc) x+bd(ax+b) (cx+d) = acx^2 + (ad +bc) x+bd ．

如果我们想要对形如 px2+qx+r px^2 +qx+r px^2 +qx+r 的二次三项式进行因式分解，那么就需要探索如下图的十字：

其中：

第一列的积 a·c=p a · c = p a · c = p 为二次项系数；

第二列的积 b·d=r b · d = r b · d = r 为常数项；

列间的交叉乘积和 a·d+b·c=q a · d +b · c = q a · d +b · c = q 为一次项系数．

则有

px2+qx+r=(ax+b)(cx+d)px^2 +qx+r = (ax+b) (cx+d)px^2 +qx+r = (ax+b) (cx+d) ．

以上方法被称为十字相乘法，这个方法可以概括为 16 个字：“头尾分解，交叉相乘，求和凑中，横向写出”．

十字相乘法的实质是凑数。
